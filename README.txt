Buat yang mau main dan belajar dengan game ini silahkan install Python dan Pygame
Cara & aturan main :
1. Gunakan panah pada keyboard untuk menggerakan player(astronout) ke kanan, kiri, atas, dan bawah
2. Tekan spasi ketika posisi astronout berada di bintang
3. Jika player terkena meteor maka posisi player akan kembali ke tempat awal dan score akan di reset
4. Ambil sebanyak mungkin bintang untuk mendapat score yang tinggi
5. Game ini bersifat endless atau tak ada habisnya maka dari itu ambilah sebanyak mungkin bintang untuk mendapat highscore baru
